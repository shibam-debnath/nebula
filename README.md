# 🌌 Nebula

**Nebula** is a next-generation code editor environment designed to treat multiple Git repositories as first-class, peer-level citizens within a single unified session. 

Stop switching windows; start managing systems.


## 🚀 The Vision
In modern development, a single "feature" often spans multiple microservices, libraries, and frontends. Traditional IDEs are built for the "Single Repo" era. **Nebula** is built for the "Universe" era.
